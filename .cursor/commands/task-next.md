# ⏭️ 下一步任務

喂！要開始下一個冒險了！挑出最重要的任務！

## 任務選擇

### 1. 📊 當前狀態評估
請分析：
- 目前完成了什麼？
- 還剩下什麼？
- 有什麼阻礙？

### 2. 🎯 下一步選擇標準

#### 優先級評分（1-10）
- **業務價值**: 對用戶的影響有多大？
- **技術依賴**: 是否阻擋其他任務？
- **風險程度**: 延遲會造成什麼問題？
- **實作難度**: 需要多少時間？
- **資源可用性**: 現在能不能做？

### 3. 🚀 建議的下一步

#### 推薦任務
```markdown
📌 任務名稱：[具體任務]
⏱️ 預估時間：[X 小時]
🎯 目標：[明確的完成標準]
⚠️ 風險：[潛在問題]
```

#### 具體步驟
1. [ ] 步驟 1：準備工作
2. [ ] 步驟 2：核心實作
3. [ ] 步驟 3：測試驗證
4. [ ] 步驟 4：文檔更新

### 4. 📋 備選任務
如果主要任務被阻塞，可以做：
- 備選 1：[任務描述] - 原因
- 備選 2：[任務描述] - 原因
- 備選 3：[任務描述] - 原因
